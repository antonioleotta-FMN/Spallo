# SPALLO

Web app Streamlit mobile-first per disegnare la spallinata sulla lenza.

## Funzioni versione prova

- input lunghezza spallinata
- input peso totale
- scelta tipo acqua
- scelta pallini disponibili
- generazione pallini a scalare
- distanze strette verso il galleggiante e più ampie verso l'amo
- schema verticale mobile-friendly
- riepilogo pallini
- download CSV

## Avvio locale

```bash
pip install -r requirements.txt
streamlit run app.py
```
